## What is this?

This is a web application template project using spring.

This is considered to be used as template project in STS 3+. Please look at [GettingStarted][1].

## Technology components
This template project uses

* Spring MVC
* JPA
* Hibernate
* Spring Data JPA
* Spring Security
* Apache Tiles
* SLF4J
* Logback
* Joda-Time
* jQuery
* Bootstrap
